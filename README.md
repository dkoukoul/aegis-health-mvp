# Aegis Health

Aegis Health is a local-first (CRDT) web app that helps health adminitrative staff manage their patients.
No data are stored in the cloud.
Data are stored locally but can be synced across devices and users.
The app is built using web technologies and can be run on any mobile device and laptop/desktop.

## MVP Features

- Patients details management
- Appointments management
- Medications management
- Health records management

## Notes

- Multilingual support (Greek and English)
- Responsive design
- Dark mode
- Offline-first (CRDT)
- Using bun TS and Svelte
